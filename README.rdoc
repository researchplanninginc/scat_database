== NOAA SCAT Database Readme

This application is a demonstration of an open-source Ruby-on-Rails application for storing and querying Shoreline Cleanup and Assessment Technique (SCAT) data collected during an oil spill.  Note that this database is compliant with NOAA's SCAT Data Standard.

Versions:

* Ruby version 2.2.4
* Rails 4.2.6



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
